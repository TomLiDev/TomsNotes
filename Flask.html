Flask

Flask is a framework, allows creation of webpages and other cool shit using python.

Jinja is a template language that can be used with Flask.

Code Syntax - {% if some.value = 2 %} - In Flask this is the syntax for if/for loops, the % sign denotes this.

Routing - Allows for connections between pages, HTML itself and other webpage elements to be created by a run.py file
